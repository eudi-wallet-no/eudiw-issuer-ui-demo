Rediger bevistype

Det kan ta inntil eitt minutt før bevisdefinisjonen er oppdatert i utstedaren.

Gyldig type for claims er string for tekst og binary for bilder. mime_type gjeld berre for binary type og må angis som mediatype ihht https://www.rfc-editor.org/rfc/rfc2397.html. Default er "image/png" for SD-JWT, attributten er ignorert for MDoc. Dersom type ikke er angitt blir type default satt til string.
